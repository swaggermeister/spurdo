Ruby Gem for English to Spurdo speak :DDD

Ruby port of the [Spurdo NPM module](https://www.npmjs.com/package/spurdo).

## Usage

```ruby
Spurdo.to_spurdo("We the People of the United States.")
# "We de beoble of de Unided States XDD"
```
